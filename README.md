# AWS Chatbot Portfolio Project

A full-stack chatbot using:

- **Next.js** frontend (Pages Router)
- Hosted on **Cloudflare Pages**
- **AWS Lambda** backend
- **AWS Bedrock** (amazon.nova-micro-v1:0)
- **API Gateway**
- **SAM** for local dev + production deployment
